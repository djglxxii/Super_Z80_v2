# Super_Z80_v2

Super_Z80_v2 is a clean reboot of a Z80-based emulator/tooling project focused on deterministic behavior, modular design, and long-running maintainability.

This repository uses AWOS-style task execution with small, bounded task packets and explicit verification per task.

OpenAI Codex CLI is the primary coding agent environment.

Repository documents are the durable source of truth; chat history is not authoritative.

Bootstrap is complete, but emulator implementation has not started yet.
